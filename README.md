## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

- Install all dependencies using the `npm install` command.
- Run the development web server using the `npm start` command.
- Run testing for helper functions using the `npm run test` command.
